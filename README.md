# skills

Personal collection of automation scripts, workflows, and pipelines.

## Workflows

| Directory | Description |
|---|---|
| [`voice-memos/`](voice-memos/) | Watches Apple Voice Memos, extracts transcripts, matches to calendar events, writes Obsidian notes |
| [`voice-memo-shortcuts/`](voice-memo-shortcuts/) | Spotlight/Raycast shortcuts to start and stop Voice Memos recordings |

## Setup

Each workflow has its own `setup.sh` and `README.md`. Dependencies are local to each directory.
